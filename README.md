# Face Recognition Attendance System (Punch In / Punch Out)

## Project Description
This project is a face-recognition-based attendance system that automatically performs
Punch In and Punch Out using real-time face detection.

The system opens a camera interface, detects a registered face, and:
- Punches IN automatically
- Waits 5 seconds
- Punches OUT automatically
- Displays records live on the UI

## Technologies Used
- Python
- Flask
- OpenCV
- HTML, CSS, JavaScript

## How to Run the Project

### Step 1: Install Python
Python 3.10+ recommended

### Step 2: Create Virtual Environment
python -m venv venv
venv\Scripts\activate

### Step 3: Install Dependencies
pip install -r requirements.txt

### Step 4: Run the Application
